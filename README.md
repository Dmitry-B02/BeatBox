BeatBox app
==============

An app from the book **'Android Programming (4th edition)'** by *Big Nerd Ranch* for sound reproduction and pitching

Learned and worked with:
-------------------------
* View customization:
  * shapes
  * items
  * layer-lists
  * selectors
* MVVM pattern
* Data Binding
* Assets
* SoundPool
